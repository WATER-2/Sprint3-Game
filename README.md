### ПВО Game
>
>Игра сделана на JAVA с использованием фреймворка LibGDX, через IDE ANDROID STUDIO.
>
>создано для финального проекта Курса «**Разработка мобильных игр**»

## О чем игра?

В игре вы должны нейтрализовать самолетики, чтобы зарабатывать очки.

В этом вам пожет *уранолитиивый боезаряд*, нейтрализующий обычные самолеты одинм залпом, а усилиные 2-мя.

У вас всего **3 жизни**, но здоровье падает с небес, иногда...

В игре нет конца, веселитесь!
